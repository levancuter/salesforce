# ⭐ 1. Chứng chỉ Platform Developer I (PD1) là gì?

**Salesforce Platform Developer I** là chứng chỉ xác nhận bạn có khả năng xây dựng ứng dụng tùy chỉnh trên Salesforce bằng **Apex, Visualforce, Lightning Web Components**, cùng với việc hiểu rõ mô hình dữ liệu, logic tự động hóa và các best practice phát triển ứng dụng.  
→ Đây là chứng chỉ **developer entry-level quan trọng nhất** trong hệ sinh thái Salesforce.

Theo nguồn tổng quan 2026, chứng chỉ PD1 xác nhận kỹ năng:

*   Xây dựng logic và UI bằng Apex và Lightning Web Components
*   SOQL/SOSL, Flow, Data Model, Trigger
*   Kiến thức triển khai (deployment), test, debug   
    Salesforce mô tả PD1 là chứng chỉ dành cho người biết phát triển logic và giao diện bằng programmatic capabilities của Lightning Platform. # ⭐ 3. Bạn cần học những gì để thi PD1?

Dựa theo exam guide và phân bổ trọng số:

## **4 domain chính của PD1**

(Nguồn 2026)  
citeturn1search1

### **1. Developer Fundamentals (≈ 23%)**

*   Kiến trúc multi-tenant
*   MVC, Lightning Component Framework
*   Declarative vs programmatic
*   Governor limits
*   Data model: object, field, relationship, External ID    [\[udemy.com\]](https://www.udemy.com/course/salesforce-certified-platform-developer-i-ii-practice-exam/)

### **2. Process Automation & Logic (Triggers, Apex)**

*   Apex class, trigger, asynchronous Apex
*   SOQL/SOSL
*   Apex best practices
*   Flow vs Apex (khi nào dùng gì)

### **3. User Interface (Lightning Web Components / Visualforce)**

*   Cơ bản về LWC
*   Event, component, data binding
*   Khi nào dùng Visualforce

### **4. Testing, Debugging & Deployment**

*   Viết test class ≥ 75% code coverage
*   Debug log, governor limits debugging
*   Deployment bằng Change Set hoặc Salesforce CLI
